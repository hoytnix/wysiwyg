## Getting Started

### Requirements

* Python 3.5+

### Installation

Anavah can be installed with pip:

```
$ pip install Anavah
```

or directly from the source code:

```
$ git clone https://github.com/pr0xmeh/anavah.git
$ cd anavah
$ python setup.py install
```

## Basic Usage

After installation, the package can imported:

```
$ python
>>> import Flask
>>> Flask.__version__
```
